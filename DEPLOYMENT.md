## 🐛 TROUBLESHOOTING

### Issue: Blank page after deployment
**Root Cause:** Base path mismatch
**Solution:**
```bash
# Verify vite.config.js
base: '/'  # Root domain deployment

# Rebuild and redeploy
npm run build
git push origin main
```

### Issue: Images not loading (404 errors)
**Root Cause:** Asset paths not using correct base
**Solution:**
- Photo.jpeg is correctly at: `/assets/photo-C7XHnt3c.jpeg`
- Rebuild: `npm run build`
- Deploy: `git push origin main`

### Issue: Styles/JS not loading
**Root Cause:** Script/style src attributes incorrect
**Verification:** Check dist/index.html
```html
<!-- Should be: -->
<script src="/assets/index-BFhS0qbq.js"></script>
<link href="/assets/index-MiZ2OJHW.css" rel="stylesheet">

<!-- NOT: -->
<script src="/portfolio/assets/index-BFhS0qbq.js"></script>  <!-- Wrong for user site -->
<link href="index-MiZ2OJHW.css" rel="stylesheet">  <!-- No leading slash -->
```

### Issue: GitHub Actions deployment fails
**Debug Steps:**
1. Check workflow logs: Actions → Deploy to GitHub Pages → Latest run
2. Look for error messages (npm install, build, or deploy failures)
3. Common fixes:
   ```bash
   # Reinstall dependencies
   rm -rf node_modules package-lock.json
   npm install
   npm run build
   git add .
   git commit -m "Fix: dependency updates"
   git push origin main
   ```

### Issue: 404 on gh-pages branch
**Solution:**
- GitHub Actions should auto-create `gh-pages` branch
- If missing, manually trigger: `npm run deploy` locally
- Or re-run GitHub Actions workflow
